Bienvenido al Manual de Ayuda

Este manual te ayudará a entender cómo usar la aplicación de pronóstico de enfermedades, que utiliza modelos avanzados como ARIMA, Holt-Winters y redes neuronales para la predicción de la mortalidad de animales.

Comenzar

¿Qué es esta aplicación?

Este sistema permite realizar predicciones sobre la mortalidad de animales, basándose en datos históricos y utilizando distintos modelos de pronóstico. Con esta herramienta podrás predecir enfermedades y gestionar mejor los recursos.

Configuración de Parámetros

A continuación puedes configurar los parámetros de la predicción. Haz clic en cada uno de ellos para ver más detalles.

Selecciona la especie de la que deseas realizar la predicción. Las opciones disponibles incluyen:

  • Avícola
  • Equino
  • Bovino
  • Caprino
  • Porcino
  • Ovino

Este parámetro afectará las predicciones basadas en los datos específicos de cada especie.

Selecciona la fecha en la que deseas que inicien las predicciones. Esto marcará el punto de partida de los datos históricos que se usarán para estimar los resultados.

Selecciona la fecha máxima hasta la cual deseas realizar la estimación. Esta fecha marcará el final del rango de predicción.

Selecciona el departamento para el cual deseas realizar la estimación. Este parámetro es crucial para filtrar los datos de manera geográfica y realizar estimaciones precisas.

Código de los Modelos

A continuación se presenta el código que implementa los modelos de ARIMA, Holt-Winters y redes neuronales.

Modelo ARIMA


# ARIMA Model
from statsmodels.tsa.arima.model import ARIMA
model = ARIMA(data, order=(5, 1, 0))
model_fit = model.fit()

Modelo Holt-Winters


# Holt-Winters Model
from statsmodels.tsa.holtwinters import ExponentialSmoothing
model_hw = ExponentialSmoothing(data, trend='add', seasonal='add', seasonal_periods=12)
model_hw_fit = model_hw.fit()

Modelo Redes Neuronales


# Neural Network Model
from keras.models import Sequential
from keras.layers import Dense, LSTM
model_nn = Sequential()
model_nn.add(LSTM(50, return_sequences=True, input_shape=(X_train.shape[1], X_train.shape[2])))
model_nn.add(LSTM(50))
model_nn.add(Dense(1))
model_nn.compile(optimizer='adam', loss='mean_squared_error')
model_nn.fit(X_train, y_train, epochs=10, batch_size=32)

Modelos de Predicción

ARIMA

ARIMA es un modelo estadístico que se utiliza para predecir series temporales, adecuado para datos con o sin estacionalidad.

Holt-Winters

El método de Holt-Winters es ideal para datos con componentes de tendencia y estacionalidad, ajustando la predicción según estos patrones.

Redes Neuronales

Las redes neuronales son un enfoque avanzado para modelar relaciones no lineales y realizar predicciones complejas basadas en grandes volúmenes de datos.

Predicción Inteligente

Combinando ARIMA, Holt-Winters y redes neuronales, nuestro sistema permite generar predicciones precisas y personalizadas para la mortalidad animal.
