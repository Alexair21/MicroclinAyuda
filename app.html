¿Qué es esta aplicación?

Esta aplicación móvil, desarrollada por el laboratorio MicroClin S.R.L., está diseñada para brindar a los usuarios la posibilidad de consultar información sobre diversas enfermedades de manera rápida y accesible.

La interfaz intuitiva permite navegar fácilmente entre las secciones principales, consultar estadísticas relacionadas con los casos totales y obtener soporte a través de un manual de ayuda. El objetivo es simplificar el acceso a información de salud confiable para pacientes, médicos y profesionales del área.

Estadísticas y Datos Clave

La aplicación presenta una sección dedicada a estadísticas generales que permite a los usuarios conocer información detallada sobre las enfermedades registradas, los casos totales y las muertes asociadas.

En la interfaz, se destacan:

  • Total de Enfermedades: El número total de enfermedades monitoreadas.
  • Casos Totales: Una visión global del impacto acumulado de las enfermedades.
  • Muertes Totales: Los fallecimientos registrados asociados a las enfermedades.

Además, incluye un apartado de "Significado de las Siglas", donde se definen las categorías de animales afectadas, tales como: avi (Aves), sui (Suinos), api (Apicultura), bov (Bovinos) y cuy (Cuyes).

Visualización de Datos y Filtrado

Esta sección de la aplicación permite a los usuarios analizar las enfermedades registradas mediante gráficos interactivos y herramientas de filtrado. Los datos se presentan de forma clara y visual, facilitando el entendimiento del impacto en diferentes categorías de animales.

Funcionalidades principales:

  • Gráficos de Barras: Representan el número de casos por categoría animal (avi, sui, bov, cuy, api).
  • Filtro Personalizado: Permite seleccionar enfermedades específicas, como la "Influenza Aviar", para visualizar casos detallados.
  • Tabla de Enfermedades Registradas: Incluye información clave como nombre, fecha, cantidad de casos y muertes asociadas.

Esta herramienta es ideal para monitorear tendencias y realizar análisis basados en datos actualizados.
